# 🤖 AI Virtual Assistant

Full-stack MERN AI VertualAssestant with AI integration

A full-stack MERN AI Virtual Assistant with voice interaction and intelligent responses using Gemini API.

---

## 🚀 Features

- 💬 AI Chat Assistant (Google Gemini API)
- 🎤 Voice Commands (Web Speech API)
- 🔐 JWT Authentication (Login/Register)
- ☁️ Cloudinary Image Upload
- ⚡ Real-time Interaction

---

## 🛠 Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other:**
- Gemini API
- Cloudinary
